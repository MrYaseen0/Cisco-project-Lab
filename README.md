# Small Office Network Project

Welcome to the **Small Office Network** project using Cisco Packet Tracer. This project simulates a small office network divided into four distinct networks:

* **Server Room**
* **Chairman Room**
* **IT Department**
* **Computer Department**

This README file will guide you through the installation process, help you get started with Cisco Packet Tracer, and provide steps for configuring end devices, switches, and routers.

---

## üì• Installation Instructions

### 1. Download Cisco Packet Tracer
Visit the official [Cisco Networking Academy](https://www.netacad.com/courses/packet-tracer) website to download Cisco Packet Tracer. You may need to create an account or log in to access the download link.

### 2. Install Cisco Packet Tracer
Follow the installation instructions provided with the downloaded file. Cisco Packet Tracer is available for various operating systems including **Windows**, **macOS**, and **Linux**.

### 3. Open the Project File
Download the `SmallOfficeNetwork.pkt` project file included in this repository. Open it using Cisco Packet Tracer.

### 4. Explore the Network
Once the project is opened, familiarize yourself with the layout and components of each network department.

## ‚öôÔ∏è Configuration Steps

### Configuring End Devices (PCs)
To configure the IP settings for the computers in the network:

1.  Click on a **PC**.
2.  Go to the **Desktop** tab.
3.  Select **IP Configuration**.
4.  Enter the appropriate **IP address**, **Subnet Mask**, and **Default Gateway**.

### Configuring the Router
To configure the router for connectivity between departments:

1.  Click on the **Router**.
2.  Go to the **CLI** tab or **Config** tab to assign IP addresses to the router interfaces.
3.  Assign **static routing** (or a dynamic protocol) to ensure packets can be sent and received across the different networks.
